Una Web Responsive sencilla que al precionar en el boton de "Agregar al Carrito" en los productos que aparecen al inicio, se añade a una lista que se presenta como un Carrito de compras.

Se puede: 

Agregar artículos a la lista.
Eliminar artículos de la lista.
Marcar artículos como comprados.
Tiene un boton para limpiar la lista.
Es responsivo.
Se utiliza funciones secillas.
Manejo Basico del DOM.
